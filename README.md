# AI_CHAT_BOARD 🤖

AI-powered chatbot built using Python and Ollama (Phi-3) for real-time conversational AI with local LLM inference.

## 🚀 Features
- Real-time chatbot interaction
- Context-aware responses
- Fast local LLM inference
- No external API dependency

## 🛠 Tech Stack
- Python
- Ollama
- Phi-3
- NLP

## ▶️ How to Run
1. Install Ollama
2. Run model:
   ollama run phi3
3. Run app:
   python app.py
